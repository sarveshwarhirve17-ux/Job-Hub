## 🚀 About The Project

Sarveshwar Social is a LinkedIn-inspired social media interface developed using pure HTML, CSS and JavaScript.

The project focuses on building a clean UI and implementing DOM manipulation concepts by allowing users to create posts dynamically.

This project was built as part of my frontend development journey to strengthen my understanding of:

- HTML Semantic Structure
- CSS Flexbox Layout
- DOM Manipulation
- Event Handling
- Dynamic Content Rendering

---

✔ Professional Navigation Bar

✔ Left Profile Sidebar
